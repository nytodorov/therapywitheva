## Customization

### Colors
The website uses CSS custom properties (variables) for easy theming. Edit the `:root` section in `styles.css`:

```css
:root {
    --primary-color: #4ECDC4;      /* Main teal color */
    --primary-dark: #3BA39C;        /* Darker teal */
    --primary-light: #A8E6E3;       /* Light teal */
    --secondary-color: #FFE66D;     /* Yellow accent */
    --text-dark: #2C3E50;           /* Dark text */
    --text-light: #546E7A;          /* Light text */
}
```
